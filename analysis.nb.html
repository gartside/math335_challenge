Math 335 Challenge Problem

Background

Early in prehistory, some descriptive names began to be used again and again until they formed a name pool for a particular culture. Parents would choose names from the pool of existing names rather than invent new ones for their children.

With the rise of Christianity, certain trends in naming practices manifested. Christians were encouraged to name their children after saints and martyrs of the church. These early Christian names can be found in many cultures today, in various forms. These were spread by early missionaries throughout the Mediterranean basin and Europe.

By the Middle Ages, the Christian influence on naming practices was pervasive. Each culture had its pool of names, which were a combination of native names and early Christian names that had been in the language long enough to be considered native.

Challenge Questions

Challenge Question One

How does your name at your birth year compare to its use historically?

Response

First we plot the number of children that were named James for each year in the data.

library(tidyverse)
#names_year <- read_csv("names_year.csv", col_types = list(year = col_date()))
names_year <- read_csv("names_year.csv")
getNameTibble <- function(x) names_year %>% filter(name==x) %>% select(c(year, Total))
james_tibble <- getNameTibble("James")
james_tibble %>% ggplot(aes(year, Total)) + geom_point() + labs(title="Frequency of children named James by year", x="Year", y="Yearly Total") + geom_vline(xintercept=1964, linetype='dashed', color='blue')

I was born in the year 1964, in which there were 41359.5 children born with the name James. It looks like the name James was most popular around 1950, but that might be due to the baby boom after World War II, and not because the name James was more popular around that time period. To check this, we should look at the proportion of the children named James each year instead of the frequency for each year.

To calculate the proportions, we subset the original data frame by year, and then add up the frequency totals for each name to get a grand total for the number of children born that year. Then we divide the yearly total for James by the grand total for the year.

getGrandTotal <- function(x) names_year %>% filter(year==x) %>% select(c(Total)) %>% sum()
james_tibble <- james_tibble %>% mutate(grandTotal = (year %>% modify(function(x) getGrandTotal(x))), propGrandTotal = Total/grandTotal)
james_tibble %>% ggplot(aes(year, grandTotal)) + geom_point() + labs(title="Total number of children born each year", x="Year", y="Total born")

james_tibble %>% ggplot(aes(year, propGrandTotal)) + geom_point() + labs(title="Proportion of children named James by year", x="Year", y="Yearly proportion") + geom_vline(xintercept=1964, linetype='dashed', color='blue')

The rise in popularity seen in the first graph from 1910 to the early 1940s is deceptive since it coincides with the growing number of births each year, which is seen in the second graph. From the third graph, we see that the popularity of the name James did not vary much between 1910 and the early 1930s, and that it most popular between the years 1936 and 1945, and has declined in popularity since 1945.

brittany_tibble <- getNameTibble("Brittany")

It looks like the popularity for the name Brittany peaked in 1990. We find this value using R:

maxBrittanyIndex <- brittany_tibble$propGrandTotal %>% which.max()
maxBrittanyYear <- brittany_tibble$year[maxBrittanyIndex]

Using R, the year that Brittany was most popular is 1990, which would make the person 31 years old. Based on the graph, I would not guess the birth year to be before 1982 nor after 2001, so I would not guess any ages below 20 and I would not guess any ages above 39.

Challenge Question Three

Mary, Martha, Peter, and Paul are all Christian names. From 1920 - 2000, compare the name usage of each of the four names.

Response:

We combine the data for the names Mary, Martha, Peter, and Paul in one table:

mary_tibble <- getNameTibble("Mary") %>% filter(year >= 1920) %>% filter(year <= 2000) %>% mutate(name = "Mary") %>% mutate(propGrandTotal = Total/(year %>% modify(function(x) getGrandTotal(x))))
martha_tibble <- getNameTibble("Martha") %>% filter(year >= 1920) %>% filter(year <= 2000) %>% mutate(name = "Martha") %>% mutate(propGrandTotal = Total/(year %>% modify(function(x) getGrandTotal(x))))
peter_tibble <- getNameTibble("Peter") %>% filter(year >= 1920) %>% filter(year <= 2000) %>% mutate(name = "Peter") %>% mutate(propGrandTotal = Total/(year %>% modify(function(x) getGrandTotal(x))))
paul_tibble <- getNameTibble("Paul") %>% filter(year >= 1920) %>% filter(year <= 2000) %>% mutate(name = "Paul") %>% mutate(propGrandTotal = Total/(year %>% modify(function(x) getGrandTotal(x))))
combined_data <- rbind(mary_tibble, martha_tibble, peter_tibble, paul_tibble)
combined_data %>% ggplot(aes(year, propGrandTotal, color = factor(name))) + geom_point() + labs(title="Proportion of children by year", x="Year", y="Yearly proportion") + scale_color_brewer(palette = "Set2")

We can see that Mary was much more popular in 1920 than Martha, Paul and Peter. The popularity of Peter has remained fairly constant, and that the popularities of Paul and Martha have decreased slowly over time. Mary has experienced a large drop on popularity from 1920 to 2000.
